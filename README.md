Final project for Web Application Development Grade.

**-- FireBreath --**

Social media, based on Twitter. In this app, users can post, follow other users and send them private messages.

This is the **backend**. Developed using:
- **.NET 8**
- **EntityFramework**
- **Microservices**
- **Repositories**
- **Units of work**
- **Middleware (ocelot)**
